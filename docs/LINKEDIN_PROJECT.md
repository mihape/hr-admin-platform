# LinkedIn Project Copy

## Project Title

HR Admin Platform - Local-first Windows desktop app for business administration

## Short Description

Built and packaged a local-first Electron Windows desktop application for small business administration workflows: invoice tracking, fleet records, attendance sheets, and utilities/rent settlements.

## Portfolio Description

HR Admin Platform demonstrates the full lifecycle around an internal operations tool, not only the UI. The project includes a classic NSIS Windows installer, release and demo build modes, local/NAS JSON data storage, backup and restore workflows, SHA256 release checksums, deployment documentation, security boundary notes, issue/PR workflow documentation, and GitHub Actions release automation design.

The app is positioned as a practical IT Support / Junior System Administrator portfolio project because it covers software packaging, Windows deployment, local data handling, backup planning, user documentation, release management, and operational trade-offs.

## Key Skills Demonstrated

- Windows desktop application packaging
- Electron app deployment
- Local-first data storage
- NAS/shared-file operational design
- Backup and restore workflow
- GitHub release management
- Documentation for users and IT support
- Security limitation documentation
- Issue, pull request, milestone, and release workflow

## Magyar Verzió

HR Admin Platform egy helyi Windows desktop admin alkalmazás számlák, flotta, jelenléti ívek és rezsi/albérlet elszámolások kezelésére.

Portfolio szempontból azt mutatja meg, hogy egy belső üzleti eszközt nem csak elkészíteni kell, hanem átadhatóan és üzemeltethetően is érdemes felépíteni: telepítő, verziózott release, demo/release build, helyi vagy NAS-os adatfájl, mentés/visszatöltés, dokumentáció, biztonsági korlátok és GitHub workflow.
