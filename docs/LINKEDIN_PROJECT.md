# LinkedIn Project Copy

## Project Title

HR Admin Platform - Windows desktop operations tool for small business administration

## Short Description

Built and packaged a local-first Electron Windows desktop app for invoice tracking, fleet records, attendance sheets, and utilities/rent settlements, with a focus on Windows deployment, release management, backup/restore, and IT support handoff.

## Portfolio Description

HR Admin Platform demonstrates the full lifecycle around an internal operations tool, not only the UI. The project includes a classic NSIS Windows installer, release and demo build modes, local/NAS JSON data storage, backup and restore workflows, SHA256 release checksums, deployment documentation, silent deployment notes, security boundary documentation, auth/RBAC roadmap, issue/PR workflow documentation, and GitHub Actions release validation.

The app is positioned as a practical IT Support / Junior System Administrator portfolio project because it covers software packaging, Windows deployment, local data handling, backup planning, user documentation, release management, supportability, and operational trade-offs.
