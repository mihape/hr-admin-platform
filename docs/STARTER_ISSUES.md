# Starter GitHub Issues

Use these as initial public issues after creating the repository.

## [Feature] Automate release build using GitHub Actions

Build Windows installers automatically on version tags, attach SHA256 checksums, and publish GitHub Release assets.

## [Docs] Create SysAdmin deployment guide and firewall prerequisites

Document installation, update flow, uninstall behavior, local data location, backup/restore, and network/firewall requirements.

## [Security] Research local data encryption with Windows DPAPI

Investigate encrypting the local JSON data file or selected sensitive fields using Windows user-bound encryption.

## [Security] Define RBAC and authentication roadmap

Document whether future versions need admin/read-only roles, Windows identity integration, or an app-level PIN.

## [Feature] Add silent installer deployment mode

Research and document command-line flags for unattended deployment through Intune, SCCM, or GPO user logon scripts.
