# Security Policy

## Supported Version

This repository currently documents and supports the latest 0.2.x release only.

## Reporting a Vulnerability

Do not open a public issue with sensitive data, real invoices, backups, or local data files.

For portfolio use, create a private note or contact the repository owner directly before sharing sensitive examples.

## Security Model

See [docs/SECURITY.md](docs/SECURITY.md) for the current local-only security model, limitations, and hardening roadmap.
